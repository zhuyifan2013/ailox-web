# Ailox — Product Document

## Vision

Build a cohesive ecosystem of AI-powered productivity apps that compound value over time. Unlike isolated tools, every Ailox app shares a unified backend and AI memory layer — meaning the more apps you use, the smarter the whole system becomes.

**Tagline:** AI-powered apps for a better life.

---

## Core Value Proposition

| Traditional Apps | Ailox |
|---|---|
| Isolated data silos | Unified cross-app AI memory |
| AI as a feature bolt-on | AI as the core architecture |
| Static functionality | Gets smarter with usage |
| Multiple subscriptions | One ecosystem |

## Target Users

**Primary:** Knowledge workers, students, and self-improvement focused individuals (25–40) who:
- Already use multiple productivity tools and feel fragmented
- Are curious about AI but want practical, daily-use value
- Value data privacy and a cohesive experience

**Secondary:** Teams and small businesses looking for a connected productivity suite.

---

## User Journey

```
Discover Ailox → Install first app (e.g. Notes) → Experience AI value →
Install second app (Goals) → AI memory connects both → Unlock cross-app insights →
Become ecosystem-dependent (high retention) → Refer others
```

## Competitive Landscape

- **Notion AI** — single-app, not ecosystem
- **Obsidian** — powerful but requires manual setup, limited AI
- **Reflect** — notes-focused, no cross-app ecosystem
- **Ailox differentiator** — the only product designed as a *connected AI ecosystem* from day one
